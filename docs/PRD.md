# STerminal 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称

STerminal

### 1.2 产品定位

一款面向开发者和运维工程师的跨桌面平台终端管理工具，集成 SSH/SFTP 客户端、本地终端管理、密钥管理、端口转发、命令片段等核心能力，提供免费云同步服务，让用户在多台设备间无缝切换工作环境。

### 1.3 目标用户

- 后端开发工程师
- DevOps / SRE 工程师
- 系统管理员
- 全栈开发者
- 需要频繁管理远程服务器的技术人员

### 1.4 支持平台

| 平台 | 最低版本 |
|------|---------|
| macOS | 11.0 (Big Sur) + |
| Windows | 10 (1903) + |
| Linux | Ubuntu 20.04 / Fedora 34 / Arch (rolling) 等主流发行版 |

### 1.5 核心原则

- 注册即可用，所有功能免费开放，无订阅、无付费墙
- 云同步免费，数据端到端加密
- 不支持团队/组织功能，仅面向个人用户
- 本地优先，离线可用，联网时自动同步

## 2. 用户账户系统

### 2.1 注册

#### 2.1.1 注册方式

| 方式 | 说明 |
|------|------|
| 邮箱注册 | 输入邮箱 + 密码，发送验证邮件，点击链接完成验证 |
| 第三方 OAuth | 支持 GitHub、Google 账号快捷注册/登录 |

#### 2.1.2 注册字段

| 字段 | 必填 | 规则 |
|------|------|------|
| 邮箱 | 是 | 合法邮箱格式，全局唯一 |
| 用户名 | 是 | 3-32 字符，字母/数字/下划线/短横线，全局唯一 |
| 密码 | 是 | 最少 8 位，至少包含大写字母、小写字母、数字各一个 |
| 确认密码 | 是 | 与密码一致 |

#### 2.1.3 验证流程

1. 提交注册表单后，向邮箱发送含 Token 的验证链接（有效期 24 小时）
2. 未验证账户可登录，但云同步功能不可用，客户端顶部提示"请验证邮箱以启用云同步"
3. 可重新发送验证邮件（60 秒冷却）

### 2.2 登录

#### 2.2.1 登录方式

| 方式 | 说明 |
|------|------|
| 邮箱 + 密码 | 标准登录 |
| GitHub OAuth | 跳转 GitHub 授权后回调登录 |
| Google OAuth | 跳转 Google 授权后回调登录 |

#### 2.2.2 登录安全

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 记住登录状态 | 勾选后本地持久化 Token，否则关闭应用后失效 | 开启 |
| Token 有效期 | 记住状态时的 Token 最长有效期 | 30 天 |
| 登录失败锁定 | 连续 N 次密码错误后锁定账户 M 分钟 | 5 次 / 15 分钟 |

### 2.3 密码管理

| 功能 | 说明 |
|------|------|
| 忘记密码 | 输入注册邮箱，发送重置链接（有效期 1 小时），点击后设置新密码 |
| 修改密码 | 设置页面，输入当前密码 + 新密码 + 确认密码 |

### 2.4 账户设置

| 配置项 | 说明 |
|--------|------|
| 修改用户名 | 3-32 字符，全局唯一检查 |
| 修改邮箱 | 需向新邮箱发送验证邮件确认 |
| 头像 | 支持上传自定义头像（JPG/PNG，最大 2MB）或使用 Gravatar |
| 删除账户 | 输入密码确认后永久删除账户及所有云端数据，本地数据保留 |
| 注销登录 | 清除本地 Token，断开云同步 |

## 3. 云同步系统

### 3.1 同步范围

以下数据均纳入云同步：

| 数据类型 | 说明 |
|----------|------|
| 主机连接 | 所有主机配置及分组结构 |
| 本地终端配置 | 所有本地终端的名称、shell、工作路径、启动脚本等 |
| 密钥对 | SSH 密钥（私钥加密存储） |
| 已知主机 | known_hosts 记录 |
| 命令片段 | Snippets 及其分组 |
| 端口转发规则 | 所有保存的转发配置 |
| 应用设置 | 主题、字体、快捷键、终端偏好等全局设置 |
| 标签/标记 | 所有标签定义及关联关系 |
| Vault 凭据 | 加密存储的密码/Token |

### 3.2 同步机制

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 自动同步 | 开启/关闭自动同步 | 开启 |
| 同步间隔 | 自动同步的轮询间隔 | 实时（WebSocket 推送 + 30 秒心跳） |
| 手动同步 | 点击工具栏同步按钮立即触发 | - |
| 冲突策略 | 同一条目在多端修改时的解决策略 | 最后修改时间优先（Last-Write-Wins） |
| 同步状态指示 | 工具栏图标显示：已同步 / 同步中 / 同步失败 / 离线 | - |

### 3.3 端到端加密

| 项目 | 说明 |
|------|------|
| 加密密钥 | 首次启用同步时由用户设置"加密口令"，本地派生 AES-256 密钥 |
| 加密范围 | 密钥私钥、密码、Vault 凭据等敏感字段在上传前客户端加密 |
| 服务端 | 服务端仅存储密文，无法解密 |
| 口令找回 | 不支持找回加密口令，遗忘后需重置同步数据 |
| 修改口令 | 设置中可修改加密口令，修改时自动重新加密所有数据 |

### 3.4 离线模式

- 未登录或网络不可用时，所有功能基于本地数据正常运行
- 离线期间的变更缓存在本地，恢复网络后自动同步
- 离线队列可在设置中查看待同步变更数量

## 4. 主机与连接管理

| 地址 | 是 | IP 地址或域名 | - |
| 端口 | 否 | SSH 端口 | 22 |
| 协议 | 否 | SSH / Mosh / Telnet | SSH |

**认证信息**

| 字段 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| 用户名 | 否 | 登录用户名，为空时连接时提示输入 | 空 |
| 认证方式 | 否 | 密码 / 密钥 / 密码+密钥 / SSH Agent / 键盘交互 | 密码 |
| 密码 | 否 | 当认证方式含密码时填写 | 空 |
| 密钥 | 否 | 从密钥管理器选择已导入的密钥，或指定本地路径 | 空 |
| 密钥口令 | 否 | 加密密钥的口令 | 空 |

**SSH 高级配置**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 启动命令 | 连接成功后自动执行的命令 | 空 |
| 环境变量 | 键值对列表，通过 SSH 协议传递到远程 | 空 |
| 字符编码 | 终端字符编码 | UTF-8 |
| Keep Alive 间隔 | 发送 keep-alive 包的间隔秒数，0 为禁用 | 60 |
| 连接超时 | 建立连接的超时时间（秒） | 10 |
| 心跳超时 | 无响应判定断连的超时时间（秒） | 30 |
| 压缩 | 是否启用 zlib 压缩 | 关闭 |

| SSH 版本 | 强制使用的 SSH 协议版本 | 自动 |
| 首选密钥交换算法 | 逗号分隔的 KEX 算法列表 | 系统默认 |
| 首选加密算法 | 逗号分隔的 Cipher 列表 | 系统默认 |
| 首选 MAC 算法 | 逗号分隔的 MAC 列表 | 系统默认 |
| 首选主机密钥算法 | 逗号分隔的 HostKey 算法列表 | 系统默认 |

**代理 / 跳板**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 代理跳转（ProxyJump） | 通过另一台主机跳转连接，从已有主机中选择 | 无 |
| ProxyCommand | 自定义代理命令（与 ProxyJump 互斥） | 空 |
| SOCKS 代理 | SOCKS5 代理地址（host:port） | 空 |
| HTTP 代理 | HTTP CONNECT 代理地址（host:port） | 空 |
| 代理认证用户名 | 代理服务器的认证用户名 | 空 |
| 代理认证密码 | 代理服务器的认证密码 | 空 |

**外观覆盖（可选，优先级高于全局终端设置）**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 终端主题 | 从主题列表中选择，或使用"跟随全局" | 跟随全局 |
| 字体 | 指定字体，或使用"跟随全局" | 跟随全局 |
| 字体大小 | 覆盖字体大小 | 跟随全局 |
| 光标样式 | Block / Underline / Bar | 跟随全局 |
| 光标闪烁 | 开启 / 关闭 | 跟随全局 |

**备注**

| 字段 | 说明 |
|------|------|
| 备注 | 多行文本，支持 Markdown，用于记录服务器用途、配置说明等 |

## 5. 本地终端管理

### 5.1 概述

本地终端允许用户在应用内直接使用本机 Shell，无需连接远程服务器。用户可以创建多个本地终端配置，每个配置可以有不同的 Shell、工作目录、启动脚本等。

### 5.2 本地终端配置

#### 5.2.1 配置字段

| 字段 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| 名称 | 是 | 终端配置的显示名称，如"项目A开发"、"Docker管理" | "本地终端" |
| 图标 | 否 | 从内置图标库中选择 | 终端默认图标 |
| 颜色标记 | 否 | 标签页/列表中的颜色标识 | 无 |
| Shell 程序 | 否 | 使用的 Shell 路径或名称 | 系统默认 Shell |
| Shell 参数 | 否 | 启动 Shell 时附加的命令行参数 | 空 |
| 默认工作路径 | 否 | 终端启动时的初始工作目录（支持浏览选择文件夹） | 用户家目录 |
| 启动脚本 | 否 | 终端就绪后自动执行的脚本内容（多行文本编辑器） | 空 |
| 启动命令 | 否 | 终端就绪后自动执行的单行命令（与启动脚本二选一） | 空 |
| 环境变量 | 否 | 额外的环境变量键值对列表，追加到系统环境变量之上 | 空 |
| 以登录 Shell 运行 | 否 | 是否以 login shell 方式启动（-l 参数） | 关闭 |

**可选 Shell 程序预设（自动检测系统可用项）**

| 平台 | 可选值 |
|------|--------|
| macOS | zsh, bash, fish, sh, tmux |
| Windows | PowerShell 7, Windows PowerShell, CMD, Git Bash, WSL (各已安装发行版) |
| Linux | bash, zsh, fish, sh, tmux |

**外观覆盖（可选，优先级高于全局终端设置）**

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 终端主题 | 从主题列表中选择 | 跟随全局 |
| 字体 | 指定字体 | 跟随全局 |
| 字体大小 | 覆盖字体大小 | 跟随全局 |
| 光标样式 | Block / Underline / Bar | 跟随全局 |
| 光标闪烁 | 开启 / 关闭 | 跟随全局 |

#### 5.2.2 启动脚本说明

- 启动脚本在 Shell 初始化完成后按行执行
- 支持多行，每行作为一条命令发送到终端
- 执行间隔可配置（行间延迟），默认 100ms
- 用途示例：
  - `cd /path/to/project && nvm use 18` - 切换目录并设置 Node 版本
  - `source ./venv/bin/activate` - 激活 Python 虚拟环境
  - `export KUBECONFIG=~/.kube/config-prod` - 设置环境变量
  - `docker-compose up -d && docker-compose logs -f` - 启动并跟踪日志

#### 5.2.3 启动脚本行间延迟配置

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 行间延迟 | 启动脚本每行命令之间的等待时间（毫秒） | 100 |

### 5.3 本地终端列表

#### 5.3.1 列表展示

- 侧边栏中与"主机"并列，独立的"本地终端"分区
- 列表项显示：图标 + 名称 + 颜色标记 + Shell 类型标识 + 工作路径摘要
- 支持拖拽排序
- 支持分组管理（与主机分组独立）

#### 5.3.2 列表操作

| 操作 | 说明 |
|------|------|
| 双击 | 使用该配置打开一个新的本地终端标签页 |
| 右键菜单 - 打开 | 新标签页打开终端 |
| 右键菜单 - 编辑 | 打开配置编辑面板 |
| 右键菜单 - 复制 | 复制一份配置 |
| 右键菜单 - 删除 | 删除该配置（确认弹窗） |
| 右键菜单 - 在新窗口中打开 | 在新的应用窗口中打开该终端 |

#### 5.3.3 默认本地终端

- 系统自带一个"默认终端"配置，使用系统默认 Shell 和家目录，不可删除
- 用户可修改默认终端的配置
- 工具栏"新建本地终端"按钮使用默认配置打开
- 快捷键 `Ctrl/Cmd + Shift + T` 使用默认配置快速打开

### 5.4 快速新建本地终端

| 功能 | 说明 |
|------|------|
| 工具栏按钮 | "+" 按钮下拉菜单中选择"新建本地终端" |
| 命令面板 | `Ctrl/Cmd + P` 打开命令面板，输入"本地终端"筛选 |
| 快捷键 | `Ctrl/Cmd + Shift + T` 打开默认本地终端 |

## 6. 终端功能

| 功能 | 说明 |
|------|------|
| 终端仿真器 | 基于 xterm.js，兼容 VT100/VT220/xterm-256color |
| 颜色支持 | 256 色 + TrueColor (24-bit) |
| Unicode 支持 | 完整 Unicode 支持，包括 CJK 字符、Emoji 的正确宽度计算 |
| 光标样式 | Block（方块）/ Underline（下划线）/ Bar（竖线） |
| 光标闪烁 | 开启 / 关闭 |
| 回滚缓冲区 | 终端回滚行数（可配置） |
| 右键行为 | 粘贴 / 显示上下文菜单（可配置） |
| 选中即复制 | 选中文本自动复制到剪贴板（可配置） |
| 鼠标事件 | 支持终端应用的鼠标事件（如 vim、htop、tmux） |
| 链接检测 | 自动识别 URL，`Ctrl/Cmd + 点击` 在浏览器打开 |
| 文件路径检测 | 自动识别文件路径，`Ctrl/Cmd + 点击` 用默认编辑器打开 |
| 括号粘贴模式 | 支持 Bracketed Paste Mode |
| 终端铃声 | 可配置：系统声音 / 视觉闪烁 / 无 |

### 6.2 终端全局配置

#### 6.2.1 外观

| 配置项 | 说明 | 默认值 |
|--------|------|--------|

### 11.2 变量系统

| 变量语法 | 说明 |
|----------|------|
| `${variable_name}` | 基本变量，执行时弹窗输入值 |
| `${variable_name:default_value}` | 带默认值的变量 |
| `${variable_name:option1\|option2\|option3}` | 下拉选择变量 |
| `${!password_variable}` | 密码变量，输入时遮蔽显示 |

执行含变量的片段时，弹出表单让用户填写所有变量的值，每个变量显示名称和输入框（或下拉框），支持记忆上次输入的值。

| 功能 | 说明 |

## 12. Vault（安全凭据库）

### 12.1 概述

Vault 是加密存储敏感信息的安全容器，用于统一管理密码、Token、API Key 等凭据。

### 12.2 Vault 解锁

| 功能 | 说明 |
|------|------|
| 主密码 | 首次使用 Vault 时设置主密码，后续每次访问需输入 |
| 自动锁定 | 超过设定时间未操作自动锁定 Vault |
| 锁定时间 | 可配置：1/5/15/30/60 分钟 / 从不自动锁定（默认 15 分钟） |
| 锁定按钮 | 手动一键锁定 Vault |

### 12.3 凭据条目

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | 是 | 凭据名称 |
| 类型 | 是 | 密码 / SSH 密码 / API Key / Token / 证书 / 自定义 |
| 用户名 | 否 | 关联的用户名 |
| 密码/值 | 是 | 凭据值，加密存储 |
| URL | 否 | 关联的 URL |
| 备注 | 否 | 补充说明 |
| 标签 | 否 | 分类标签 |
| 过期时间 | 否 | 凭据过期时间，到期提醒 |

### 12.4 凭据操作

| 功能 | 说明 |
|------|------|
| 密码生成器 | 内置密码生成器，可配置长度、字符类型（大写/小写/数字/特殊字符）、排除易混淆字符 |
| 复制密码 | 一键复制，剪贴板 N 秒后自动清除（可配置：10/30/60/120 秒，默认 30 秒） |
| 显示/隐藏 | 密码默认遮蔽，点击眼睛图标切换显示 |
| 关联主机 | 凭据可关联到主机配置，自动填充密码 |
| 搜索 | 搜索凭据名称、用户名、备注、标签 |
| 分组 | 凭据支持文件夹分组 |
| 导出 | 加密导出为 JSON（需输入导出密码） |
| 导入 | 从加密 JSON 导入（需输入导出密码） |
